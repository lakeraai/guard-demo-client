**Format:** `(?i)\b(company1|company2|...)\b` — case-insensitive, whole-word match.

## Usage

1. Go to [platform.lakera.ai](https://platform.lakera.ai) → **Policies** → your policy → **Edit**
2. Open **Advanced settings** → **Custom detectors**
3. Add a new detector for each regex above
4. Set action: **Mask** or **Block** as needed
5. With `on_flagged: mask` in LiteLLM config, flagged content is masked instead of blocked

## Notes

- Each regex is ~500 companies; 12 regexes ≈ 6,000+ unique enterprises
- Some names appear in multiple regexes (deduplication optional)
- Names with spaces use `\s` or `.?` (e.g., `bank of america`, `procter.?gamble`)
- Escape special regex chars: `j\.?d\.?com` for jd.com
